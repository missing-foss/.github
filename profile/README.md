# Missing FOSS

Software built for one household's real daily use, published in case it's
useful to yours. No telemetry, no CDN, no growth plans - just tools that
had to work.

## Trobar

Self-hosted music library sync: pick artists, albums, or playlists in a web
app and keep them offline on Android devices, DAPs, SD cards, or any folder.
Your files, byte-for-byte (or transcoded to MP3 when a small card asks
for it).

- [trobar-server](https://github.com/missing-foss/trobar-server) - the server + web UI (AGPL-3.0)
- [trobar-android](https://github.com/missing-foss/trobar-android) - Android client (GPL-3.0)
- [trobar-desktop](https://github.com/missing-foss/trobar-desktop) - desktop client for SD cards & USB storage (GPL-3.0)

*Trobar* is Occitan for "to find" - the verb the troubadours got their name
from. The full story is in the
[server repo's docs](https://github.com/missing-foss/trobar-server/blob/main/docs/why-trobar.md).
